Cloud Identity	Foydalanuvchi identitetini boshqarish	Single Sign-On (SSO), multi-factor authentication (MFA)

3. Data Protection & Encryption

Cloud Data Loss Prevention (DLP)	Maxfiy ma’lumotlarni himoya qilish	Shaxsiy ma’lumotlarni aniqlash va maskalash
